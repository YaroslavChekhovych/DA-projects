# RF Sessions Analysis — Аналіз сигналів та поведінкових патернів

## Огляд

Аналіз RF-сесій із реальної системи моніторингу радіосигналів, спрямований на виявлення закономірностей у використанні частот, поведінці протоколів та характеристиках сигналів.

Проєкт демонструє повний аналітичний цикл — від обробки сирих логів і реконструкції сесій до аналізу даних та формування інсайтів.

## Аналіз

### Розподіл протоколів
- Визначено домінуючі RF-протоколи  
- Оцінено повторюваність сигналних патернів  

### Поведінка частот
- Проаналізовано розподіл частотних діапазонів  
- Розділено вузькосмугові та широкосмугові сигнали  

### Активність за частотами
- Виявлено концентрацію активності в окремих діапазонах  
- Визначено типові робочі частоти  

### Активність джерел
- Проаналізовано розподіл сесій між джерелами даних  
- Оцінено навантаження та покриття системи  

### Поведінкові патерни
- Виявлено повторювані характеристики сигналів  
- Визначено кластеризацію за частотами та шириною діапазону  
- Розділено стабільні та адаптивні патерни  

---

## Ключові інсайти

- Активність сигналів концентрується в окремих частотних діапазонах  
- Поведінка сигналів є структурованою та повторюваною  
- Виявлені патерни можуть бути використані для:
  - класифікації  
  - виявлення аномалій  
  - побудови аналітичних моделей
